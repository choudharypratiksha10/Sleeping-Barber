A barber shop(saloon) has one barber, one barber chair and N chairs for waiting customers to sit. If there is no customer, the barber goes to sleep. When a customer arrives, he has to wake up the sleeping barber. If additional customers arrive while the barber is cutting a customer’s hair then they sit if chairs are empty, they leave the shop. Customers are serviced in FCFS order. The problem is to program the barber and customers without getting into race condition.
